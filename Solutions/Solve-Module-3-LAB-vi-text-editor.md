# เฉลยลำดับการกดปุ่ม (Key Sequences):

1. `vi /tmp/vi_lab.txt` [Enter]

2. `/down` [Enter] -> กด `cw` แล้วพิมพ์ `up` [Esc]

3. `/192` [Enter] -> กด `cw` แล้วพิมพ์ `10.0.0.1` [Esc]

4. `G` -> กด `o` (ขึ้นบรรทัดใหม่ด้านล่าง) -> พิมพ์ `GATEWAY=10.0.0.254` [Esc]

5. `gg` -> กด `dd`

6. พิมพ์ `:wq` [Enter]
