# แบบฝึกหัด: การติดตั้งและกำหนดค่า LAMP Stack บน RHEL

**วัตถุประสงค์:** เพื่อฝึกฝนทักษะการจัดการ Package ด้วย DNF และการบริหารจัดการ Service ด้วย systemctl บนระบบปฏิบัติการ RHEL

---

## สถานการณ์จำลอง (Scenario)
คุณได้รับมอบหมายให้เตรียมเครื่องเซิร์ฟเวอร์ RHEL 8/9 เพื่อรองรับการทำงานของเว็บไซต์ที่เขียนด้วยภาษา PHP และต้องมีการเชื่อมต่อกับฐานข้อมูล MariaDB (MySQL) โดยมีเงื่อนไขด้านความปลอดภัยและการตั้งค่าระบบพื้นฐาน

## โจทย์การปฏิบัติ (Lab Steps)

### ส่วนที่ 1: การจัดการซอฟต์แวร์ (Package Management)
1. ค้นหา Package ที่เกี่ยวข้องกับ `mariadb-server` เพื่อยืนยันชื่อที่ถูกต้องใน Repository
2. ติดตั้งซอฟต์แวร์ที่จำเป็นทั้งหมดในคำสั่งเดียว ประกอบด้วย:
   - Apache Web Server (`httpd`)
   - MariaDB Server (`mariadb-server`)
   - PHP (`php`)
   - PHP Extension สำหรับ MySQL (`php-mysqlnd`)

### ส่วนที่ 2: การบริหารจัดการระบบ (Services Management)
3. สั่งให้ Service `httpd` และ `mariadb` เริ่มทำงาน (Start) ทันที
4. ตั้งค่าให้ Service ทั้งสองตัวรันอัตโนมัติทุกครั้งที่เปิดเครื่อง (Enable)
5. ตรวจสอบสถานะ (Status) ของทั้งสอง Service ว่ารันอยู่ปกติหรือไม่

### ส่วนที่ 3: การตั้งค่าระบบเครือข่ายและไฟล์ (System Configuration)
6. กำหนดค่า Firewall ให้เปิดใช้งาน Service `http` (Port 80) แบบถาวร (Permanent) และทำการโหลดค่าใหม่ (Reload)
7. สร้างไฟล์ทดสอบชื่อ `info.php` ไว้ที่พาธ `/var/www/html/` โดยให้แสดงผลฟังก์ชัน `phpinfo();`
8. (โบนัส) ใช้คำสั่งพื้นฐานเพื่อเพิ่มความปลอดภัยให้ MariaDB หลังการติดตั้ง
