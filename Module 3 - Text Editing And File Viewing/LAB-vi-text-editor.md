# โจทย์ปฏิบัติการ Vi
### ขั้นเตรียมไฟล์ (Run ใน Terminal):
```bash
cat <<EOF > /tmp/vi_lab.txt
# RHEL Configuration Test
INTERFACE=eth0
STATUS=down
IP=192.168.1.100
NETMASK=255.255.255.0
EOF
```

### โจทย์
1. เปิดไฟล์: `/tmp/vi_lab.txt`
2. แก้ไขค่า: เปลี่ยน `STATUS=dow` เป็น `STATUS=up`
3. ค้นหา: ค้นหาคำว่า `192.168.1.100` แล้วเปลี่ยนเป็น `10.0.0.1`
4. เพิ่มข้อมูล: ไปที่บรรทัดล่างสุดของไฟล์ แล้วพิมพ์คำว่า `GATEWAY=10.0.0.254`
5. ลบข้อมูล: ลบบรรทัดแรกที่มีเครื่องหมาย `#` ทิ้งไป
6. บันทึก: บันทึกและออกจากโปรแกรม
